# BPB Panel v4.6 (UI Concept / Refactored)

这是一个用于 Cloudflare Worker 节点管理的后台管理面板**概念设计**。

## ⚠️ 重要声明：这是一个纯前端项目

请在下载或使用前仔细阅读这一点：本项目目前**仅包含用户界面 (UI)** 和前端交互逻辑。它**不包含**任何后端代码、数据库连接或真实的节点管理功能。

* 点击“扫描 IP”不会真的去扫描 Cloudflare 的 IP 段。
* 点击“保存设置”只会弹出一个成功的提示框，不会真的修改服务器配置。
* 登录界面是模拟的，输入任何密码都可以进入。

这个项目的目的是提供一套结构清晰、易于维护的前端模版。

## 项目特点

1. **原生开发，结构分离**
代码已经被拆分为 `index.html` (结构)、`style.css` (样式) 和 `script.js` (逻辑)。这与某些将所有代码塞进一个文件的版本不同，更适合开发者进行阅读和二次开发。
2. **多语言支持 (i18n)**
内置了一套轻量级的前端翻译引擎。支持英语、简体中文、俄语和波斯语。
* 特别优化了**波斯语 (Farsi)** 的支持：当切换到波斯语时，整个布局会自动镜像翻转为 RTL (从右向左) 模式。
* 包含平滑的语言切换过渡动画。


3. **视觉设计**
* **深色模式**：默认采用深蓝/黑色的科技风格配色。
* **响应式布局**：完美适配手机端。在移动设备上，侧边栏会自动折叠。
* **易于换肤**：所有的颜色都定义在 CSS 变量中，改几个 hex 值就能换整套主题。


4. **交互演示**
包含了一些模拟脚本来展示 UI 交互效果：
* **IP 扫描器**：模拟终端滚动效果。
* **AI 混淆**：全屏矩阵动画演示。
* **Toast 通知**：无依赖的手写通知组件。



## 文件结构

* `index.html` - 页面骨架。
* `style.css` - 样式表。
* `script.js` - 交互逻辑与模拟数据。

## 如何运行

因为没有后端依赖，你不需要安装 Node.js，也不需要配置 Nginx。

1. 下载本项目的所有文件。
2. 直接双击打开 `index.html`。
3. 或者使用 VS Code 的 "Live Server" 插件运行（推荐）。

## 开发者自定义指南

如果你想把这个 UI 对接到真实的后端：

1. 打开 `script.js`。
2. 搜索 `handleLogin` 函数，替换掉里面的模拟逻辑，接入你的鉴权 API。
3. 搜索 `saveSettings` 函数，将它改为向你的 API 发送 POST 请求。
4. 搜索 `translations` 对象，可以随意修改或添加新的翻译文案。
